Burmese Food in Chiangmai 🌶️

A minimalist and professional web directory designed to help food lovers discover authentic Burmese flavors in Chiang Mai, Thailand.
🚀 Features

    Dynamic Shop Directory: Interactive grid layout displaying local shops with images and links.

    Categorized Navigation: A smart "Shop List" dropdown that allows users to jump directly to specific restaurant cards.

    Responsive Design: Fully optimized for desktops, tablets, and mobile devices.

    Tech-Focused Aesthetic: Minimalist design with a modern "line and wash" feel and smooth scroll interactions.

📂 Project Structure

├── index.html # Main website structure
├── style.css # Global styles and CSS variables
├── script.js # Dynamic content generation and scroll logic
└── static/ # Assets folder
├── Logo.png # Brand identity
├── favicon.png # Browser tab icon
└── [shop-images] # High-quality photos of local restaurants

🛠️ Technical Implementation
Dynamic Data Management

The project uses a centralized shops array in script.js. This allows for easy maintenance—adding a new shop is as simple as adding a new object to the array.
Custom UI Components

    Scroll Logic: Implemented scroll-behavior: smooth in CSS for seamless navigation.

    Scroll Bar Customization: The dropdown menu features a custom-styled scroll bar to match the "Tech Minimalist" theme.

    Lazy Loading: Images are optimized for performance using native browser lazy loading.

🎨 Design System

    Primary Colors: Dark Navy (#003049), Deep Red (#d62828), and Golden Yellow (#fcbf49).

    Typography: Utilizing the "Inter" font family for high readability and a modern tech look.

    Iconography: Powered by Font Awesome 6.0.0
